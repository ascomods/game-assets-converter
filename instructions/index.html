Instructions

This tool currently provides the ability to replace and edit model parts from models of the DB Raging Blast series.
To use it, simply launch the program click on the open button and provide the uncompressed SPR and IORAM files (so technically STPK files).

Once the files are loaded, the program lists all model parts it has found.

You can select one or multiples parts and do one of the following actions :

  • Open: Load the SPR and IORAM files

  • Save: Overwrite the currently loaded SPR and IORAM files

  • Save As: Choose a location to save currently loaded SPR and IORAM files

  • Disable Parts: Disable the selected model parts (basically 00 out the model part)

  • Import: Import and replace currently selected model parts

  • Export: Export currently selected model parts

Here I've selected Goku's face and hair (000_goku_0_1p_PS3) and exported them.

You will get a folder for each parts and they will contain a file named data.obj, which can be edited in blender, and another file called data.txt which has the information that the OBJ format doesn't support and that are required for the model part to be imported back into the game.
You can manually edit the information in data.txt if you want. A better support is planned in the future by using the FBX format.

As you can see I didn't include the normals in the OBJ file and I've put them instead in the data.txt file. That's because normals get recalculated by blender when exporting the model after edition. So I left it like that for now... So let's load the models parts we've extracted into blender.

Let's apply the same process as before to Broly (069_brolly_2_1p_PS3) to get his hair and replace Goku's hair.

As you can see the hair is not aligned properly at all, so we need to move broly's hair to the right position. In order to do that we need to be in "Edit mode" and not in "Object mode" as we want to move the position of all vertices. So switch to "Edit mode" (usually TAB as a shortcut) and move and resize the hair to your liking. You can even edit the vertices if you want but it is experimental for now...

After your done with your modifications, export the model part as "data.obj" and replace the one in Broly's hair folder.
Indeed, it must be in the same folder as the data.txt that concerns broly's hair. After that, rename the folder containing the modified hair to match the name of the part you would like to replace by importing the new one (here it is polySurfaceShape697-HAIR_ for Goku's hair).

After selecting the right part you can click on the import button and select the folder that contains your modified part. The program will try to see if, for each selected model part, there is a folder with the same name inside the folder you have provided. It will ignore other folders (here blabla is ignored) but it won't tolerate if a selected part doesn't have a matching folder. This could work for multiple parts import as well but there seems to be a bug right now, so I recommend only importing parts one by one.

Now you can either overwrite the loaded SPR and IORAM STPK files with the save button, or save them as separated files with the save as button.

You can now recompress the files with dbrb_compressor or put them directly into the AFS file. Indeed, I usually increase the size of the AFS slot and put the uncompressed SPR and IORAM STPK files into the right slots. If, on the other hand, your files are smaller than the current slot size, just fill the bottom of the files with 00 to avoid in-game crash. You can achieve that pretty much easily in 010 editor by going into "Edit > Insert/Overwrite > Insert bytes" and selecting a specific range, starting at the end of the file and ending at the wanted slot max size.
